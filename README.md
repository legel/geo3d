# Geo3D

Combining Geospatial Information with 3D digital twins.



This README is to be updated. Current inclusions:
1. OPF scripts to use pyopf to convert Pix4D OPF projects into colmap/nerfstudio formats
2. Reduced size nerfstudio library with changes to work with python 3.11


Installation process
1. Make conda environment with python 3.11
2. Install nerfstudio (and verify with `ns-train splatfacto --help`)
3. Install dn-splatter (and verify with `ns-train dn-splatter --help`)
4. Install pyopf
